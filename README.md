# Uncovering STEAP3 as a Ribosome-Associated Protein upon Erythroid Maturation of Murine Erythroleukemia Cells 
This repository contains code and data to reproduce the analysis presented in the paper [Paper final title](Paperlink).

- 📂 ``Count_files``: Count quantification files for all samples in the study.
- 📂 ``MEL_exploratory_plots``: Plots prdocued during exploratory data analysis.
- 📂 ``DGEA``: Differential Gene Expression Analysis (DGEA) output.
- 📂 ``Pathways``: Results from Over-Representation Analysis (ORA), Gene Set Enrichment Analysis (GSEA) and Active Subnetwork Analysis (ASA) with ``pathfindR``.
- ``1. DGEA_and_plots.R``: Code for exploratory analysis and DGEA.
- ``2. Pathways.R``: Code for ORA, GSEA and ASA
